## üëã Welcome!

**The ToDo App** is a simple, beautiful, and fast mobile todo application built with React Native and Expo. Easily manage your daily tasks, filter by status, and enjoy persistent storage. Perfect for learning and productivity!

---

## üöÄ Features
- Add, edit, and delete tasks
- Filter tasks: All, Completed, Pending
- Persistent data with AsyncStorage
- Fast state management with Zustand
- Clean UI with NativeWind (Tailwind CSS)
- Light & dark theme support

## üì± How to Run (Expo Go)

1. **Install [Expo Go](https://expo.dev/client) on your mobile device**
2. **Clone this repo:**
   ```sh
   git clone https://github.com/Nappu-Ghost/todo.git
   cd todo
   ```
3. **Install dependencies:**
   ```sh
   npm install
   ```
4. **Start the development server:**
   ```sh
   npx expo start
   ```
5. **Scan the QR code** in your terminal or browser using Expo Go app
6. **Enjoy!** üéâ
